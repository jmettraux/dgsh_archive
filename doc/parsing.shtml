Parsing stat blocks

Dgsh ships with a stat block parsing utility named Raptor. It also ships with stat block examples stored in the 'sbo' folder.

./raptor.sh -f sbo/standard1.sbo

or

raptor -f sbo\standard1.sbo

will turn the file sbo/standard1.sbo into a file named ugolin.xml

Note:
The sbo folder and its content will appear in dgsh-0.6.03. Here are standard1.sbo and standard2.sbo anyway.
